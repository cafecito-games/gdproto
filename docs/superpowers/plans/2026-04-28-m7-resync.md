## Self-Review

**Risks:**

1. **Empty-line interleaving.** The new format has subtle blank-line patterns (especially around the wrapper class name and between top-level items). Iterate against the golden until exact.
2. **Float formatting.** Same risk as M5; Go's `strconv.FormatFloat` may not match Python's `str(float)` for some values. The fixture only uses simple values; if not, fall back to a custom formatter.
3. **`from_text_format` complexity.** Largest new method. Don't try to write it from scratch; port Python verbatim, building the same `MatchStatement` / `IfStatement` / `ForStatement` trees.
4. **`Oneof` accessors.** The setter side has subtle behavior: setting `_email` clears `_discord` (and vice versa) AND updates `_oneof_contact`. The cross-clearing logic must come before the assignment, not after.
